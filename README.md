# Listing Promo Banners

This module is use to display catchy promo banners on corresponding category/product-listing pages to let customers be aware of all hot deals and specials offers. Draw visitors' attention to specific products to trigger impulse purchases and increase an average order value.

## Installation

```bash
composer config repositories.hitarthpattani-git vcs https://github.com/hitarthpattani/module-listing-promo-banners.git
composer require hitarthpattani/module-listing-promo-banners:dev-master
bin/magento setup:upgrade
```
